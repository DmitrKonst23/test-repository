# Тестовый проект для изучения Git

Простой практический проект, чтобы освоить базовые концепции и команды Git. А также изучить основы работы с удаленным репозиторием.

## Цели проекта
- Научиться работать с основными командами Git
- Понять принципы ветвления и слияния
- Освоить работу с удалёнными репозиториями
- Научиться работать с GitHub
